Master's Thesis in Business Intelligence and Data Mining

My master's thesis focuses on designing and developing an advanced data collection program aimed at digital content generation companies. The primary goal of this project is to identify improvement opportunities and streamline creative processes, which is essential in a constantly evolving market.

Using Python and several of its libraries, I have developed a system that allows for efficient data collection and analysis. Database management is handled with SQL, ensuring the integrity and accessibility of stored data. For advanced data visualization and analysis, I use PowerBI, a powerful tool that facilitates the creation of interactive reports and dashboards, enabling companies to make informed decisions.

Currently, I am working on automating the data collection process to be performed daily. This automation not only optimizes time and resources but also ensures the availability of up-to-date data for continuous analysis. The system is designed to be scalable and adaptable to different needs and data volumes, making it a robust solution for the digital content generation sector.

This thesis project combines theory and practice in business intelligence and data mining, demonstrating how modern technologies can be integrated to significantly improve business processes and decision-making.

Technologies used: Python, SQL, PowerBI

API Development for Rural School

The API Development project for a rural school using Python, FastAPI, and Open Meteo. This project is a comprehensive initiative to manage student attendance and manage key user profiles within the school environment. The API enables efficient student attendance tracking and provides specific functionalities for different roles within the school.

Using FastAPI, a RESTful API is developed that integrates attendance recording, user management, and authentication functionalities. Open Meteo is used to integrate weather data that may be useful for school-related decisions, such as weather-related suspensions. This ensures that school administration can make informed decisions based on weather conditions.

User profiles, including teachers, mentors, students, and administrators, have differentiated access through the API. Teachers can take attendance, mentors can manage discipline and track academic progress, students can access their records, and administrators have full control over the platform. This personalized approach ensures that each user has access to the tools and data necessary for their specific responsibilities.

This project combines modern web development technologies with the specific needs of a rural school, providing an effective and scalable solution for school management and automated attendance tracking. The implementation of this API not only improves operational efficiency but also facilitates a more organized and data-driven educational environment.

Technologies used: Python, FastAPI, Open Meteo, RESTful API

PokeAPI usage to collect data

The Data Loading from PokeAPI project using Python, PokeAPI, Django, and Bootstrap is a web application that allows users to explore detailed information about Pokémon using data obtained from the PokeAPI. The application is developed with Django, a Python web framework, and enhanced with Bootstrap for a modern and responsive design, ensuring an attractive and accessible user experience.

By integrating the PokeAPI, the application can retrieve accurate data about Pokémon "berries," including details such as types, abilities, statistics, and more. Users can search and filter information according to different criteria and view detailed information. This interactive functionality allows users to obtain valuable and entertaining information efficiently.

The user interface is created using Bootstrap to ensure an engaging and user-friendly experience on both desktop and mobile devices. Users can explore and learn interactively and educationally, enriching the user experience by providing an easy-to-navigate and visually pleasing platform.

This project demonstrates how to use technologies such as Python, Django, Bootstrap, and external APIs to create a dynamic and fun web application centered around Pokémon data, providing an interactive experience for Pokémon enthusiasts and developers alike. The implementation of these technologies showcases the ability to build robust and scalable applications that can handle large volumes of data and offer rich functionality.

Technologies used: Python, PokeAPI, Django, Bootstrap

Titanic data analysis

The Titanic Data Study project is an exploration and analysis of data using machine learning techniques to study the Titanic tragedy. A dataset available on Kaggle was used, containing detailed information about Titanic passengers, including details such as age, gender, ticket class, fare paid, and embarkation port.

With Python, Pandas, and Scikit-learn, a comprehensive analysis of the data was conducted to better understand the patterns and factors that contributed to the survival or death of the passengers. Techniques for data cleaning and preparation, data visualization, and predictive modeling using machine learning algorithms such as decision trees, random forests, and logistic regression were applied. These techniques allowed for extracting valuable insights and establishing correlations between various variables in the dataset.

The main goal was to build a predictive model that could estimate the probability of a passenger's survival based on their individual characteristics. This included exploring different variables and scenarios to understand how factors such as gender, age, and ticket class influenced the chances of surviving the Titanic disaster. The detailed analysis of these factors led to the development of a robust and reliable model.

The application of machine learning techniques allowed for significant analysis and conclusions about this historical event, providing valuable insights into the factors that might have influenced survival rates in different situations. This project illustrates how data analysis and machine learning tools can be applied to historical datasets to gain insights and make useful predictions, demonstrating the power of these technologies in analyzing past events.

Technologies used: Python, Pandas, Scikit-learn, Kaggle
