Proyecto 1: Desarrollo de API para escuela rural

El proyecto de Desarrollo de API para una escuela rural con Python, FastAPI y Open Meteo es una iniciativa integral para gestionar la asistencia de los alumnos y administrar perfiles de usuarios clave en el entorno escolar. La API permite registrar la asistencia de los alumnos de manera eficiente y proporciona funcionalidades específicas para diferentes roles dentro de la escuela.

Utilizando FastAPI, se desarrolla una API RESTful que integra funcionalidades de registro de asistencia, gestión de usuarios y autenticación. Open Meteo se utiliza para integrar datos meteorológicos que pueden ser útiles para decisiones relacionadas con la escuela, como suspensiones por mal tiempo.

Los perfiles de usuarios, incluidos profesores, preceptores, alumnos y administradores, tienen acceso diferenciado a través de la API. Los profesores pueden tomar la asistencia, los preceptores pueden gestionar la disciplina y el seguimiento del progreso académico, los alumnos pueden acceder a sus registros y los administradores tienen control total sobre la plataforma.

Este proyecto combina tecnologías modernas de desarrollo web con las necesidades específicas de una escuela rural, brindando una solución efectiva y escalable para la gestión escolar y la toma de asistencia automatizada.

Proyecto 2: Carga de datos desde PokeAPI

El proyecto de Carga de datos desde PokeAPI utilizando Python, la API PokeAPI, Django y Bootstrap es una aplicación web que permite a los usuarios explorar información detallada sobre Pokémon utilizando datos obtenidos de la API PokeAPI. La aplicación se desarrolla con Django, un marco web de Python, y se mejora con Bootstrap para un diseño moderno y receptivo.

Mediante la integración de la API PokeAPI, la aplicación puede recuperar datos precisos sobre Pokémon, incluidos detalles como tipos, habilidades, estadísticas y más. Los usuarios pueden buscar y filtrar Pokémon según diferentes criterios y visualizar información detallada sobre cada uno.

La interfaz de usuario se crea utilizando Bootstrap para garantizar una experiencia atractiva y fácil de usar en dispositivos de escritorio y móviles. Los usuarios pueden explorar y aprender sobre diferentes Pokémon de manera interactiva y educativa.

Este proyecto muestra cómo utilizar tecnologías como Python, Django, Bootstrap y APIs externas para crear una aplicación web dinámica y divertida centrada en datos de Pokémon, proporcionando una experiencia interactiva para los entusiastas de Pokémon y desarrolladores por igual.

Proyecto 3: Estudio de datos del barco Titanic

El proyecto de Estudio de Datos del barco Titanic es una exploración y análisis de datos utilizando técnicas de machine learning para estudiar la tragedia del Titanic. Se utilizó una base de datos disponible en Kaggle que contiene información detallada sobre los pasajeros del Titanic, incluidos detalles como edad, género, clase de boleto, tarifa pagada, embarque, etc.

Con Python, Pandas y Scikit-learn, se realizó un análisis exhaustivo de los datos para comprender mejor los patrones y factores que contribuyeron a la supervivencia o fallecimiento de los pasajeros. Se aplicaron técnicas de limpieza y preparación de datos, visualización de datos y modelado predictivo utilizando algoritmos de machine learning como árboles de decisión, bosques aleatorios o regresión logística.

El objetivo principal fue construir un modelo predictivo que pudiera predecir la probabilidad de supervivencia de un pasajero en función de sus características individuales. Esto incluyó explorar diferentes variables y escenarios para entender cómo factores como género, edad, clase de boleto, etc., influyeron en las posibilidades de sobrevivir al desastre del Titanic.

La aplicación de técnicas de machine learning permitió analizar y sacar conclusiones significativas sobre este evento histórico, proporcionando información valiosa sobre los factores que podrían haber influido en las tasas de supervivencia en diferentes situaciones. Este proyecto ilustra cómo las herramientas de análisis de datos y machine learning pueden aplicarse a conjuntos de datos históricos para obtener información y realizar predicciones útiles.
